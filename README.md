# 🌙 Telegram Night Link Cleaner

A simple and effective Telegram bot designed to restrict unwanted links in your group chats.

## 🚀 What it does
This bot actively monitors your Telegram group and automatically deletes messages that contain unwanted external links (such as YouTube videos or other Telegram channel links). When a link is detected and removed, the bot leaves a clean placeholder indicating that a message was deleted, keeping your community safe from spam.

## 🛠️ Tech Stack & Setup
This project is built with Python and is ready for easy cloud deployment:
* **Core Logic:** Powered by Python in `bot.py`.
* **Dependencies:** Easily install required packages using `requirements.txt`.
* **Deployment:** Includes a `procfile` for quick deployment to platforms like Heroku or Render.
